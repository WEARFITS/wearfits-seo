Pricing FAQ

What counts as an API call?

One API call = one product try-on session created. Multiple views of the same session don't count as extra calls.

Can I upgrade or downgrade my plan?

Yes! Change plans anytime. Upgrades are instant. Downgrades take effect at next billing cycle. Pro-rated refunds available.

What happens if I exceed my API call limit?

Free tier: API returns 429 (rate limit) errors. Pro tier: Automatically billed $0.002/call for overages. You can set hard limits in dashboard.

Do you offer annual billing?

Yes! Pay annually and save 20%. Pro plan: $950/year (vs $1188 monthly). Contact sales for Enterprise annual contracts.

Is there a free trial for Pro?

Absolutely! 14-day free trial. No credit card required. Full access to all Pro features.
